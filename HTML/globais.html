Atributo Class
Texto que aparece no Preview referente a classe que será utilizada no JavaScript.

Atributo Contenteditable
Atributo que permite alteração do texto no Preview. As alterações só ficam salvas enquanto o arquivo estiver aberto.

Atributo Data-*
É um atributo utilizado na conexão de páginas da web com scripts ou linguagens. A separação entre nomes ocorre através do hifen(-),

Atributo Hidden
Atributo que esconde/oculta um elemento, atributo ou texto no Preview.

Atributo ID e Style
Atributo ID define um identificador (de preferência, único em todo documento) que será usado para linkar, scriptar ou estilizar usando CSS ou o JavaScript. É possível estilizar a core da fonte no HTML pelo atributo STYLE.

Atributo Tabindex
Atributo que define a possibilidade e a ordem de tabulação, para facilitar a navegação num site utilizando o teclado.

Atributo Title
Atributo que permite a criação de um título vinculado à tag que só aparece no Preview, após deixar a seta do mouse "descansando" no elemento.
